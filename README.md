<h1>DevLabs Code Conventions</h1>
========================


<h2>Основни настройки на Code Editor-а:</h2>
========================
<ul>
    <li>4 space-чета за indentation</li>
    <li>"indentation using spaces", а не "using tabs".</li>
    <li>Line endings: Unix</li>
</ul>
